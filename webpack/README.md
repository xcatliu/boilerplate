# Boilerplate Basic

> For static one page website.
> With modular JavaScript built by Webpack.
> You can use most modules from npm.

## Technology Stack

- Basic HTML, CSS
- Webpack
- ESLint airbnb/legacy

## Getting Start

1. `npm install`
2. `npm start`
